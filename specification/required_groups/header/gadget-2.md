For backwards compatibility reasons, the snapshot’s header format also includes all fields specified by the  [Gadget-2 header](https://wwwmpa.mpa-garching.mpg.de/gadget/users-guide.pdf).
Some of these fields may contain duplicate information to other fields; where possible, prefer the use of the other fields, as they will be more informative (have descriptive names, units, etc.) and/or be more relevant.
The mapping for the Gadget-2 fields is provided below; a number in brackets represents that the field is an array of length number.
Note that since GUSTEAU snapshots use virtual datasets, all particles are considered to be contained in single files, regardless of the actual file contents.

| Name                | Type      | Mapping                                              | Description if different from mapping                                                                                                     |
| ------------------- | --------- | ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| NumPart_ThisFile    | uint32[6] | [/Header.ParticleNumbers](#header-particlenumbers)   | The least significant word of the number of the 6 particles mapped to PartTypeX. Identical to NumPart_Total since we use virtual datasets |
| MassTable           | double[6] | [/Header.InitialMassTable](#header-initialmasstable) | Mean initial mass of the 6 particles mapped to PartTypeX. If set to 0, use the `Dataset` in the corresponding group                       |
| Time                | double    | [/Header.Time](#header-time)                         |                                                                                                                                           |
| Redshift            | double    | [/Header.Redshift](#header-redshift)                 |                                                                                                                                           |
| Flag_Sfr            | int32     |                                                      | Unused, set to 0                                                                                                                          |
| Flag_Feedback       | int32     |                                                      | Unused, set to 0                                                                                                                          |
| NumPart_Total       | int32[6]  | [/Header.ParticleNumbers](#header-particlenumbers)   | The least significant word of the number of the 6 particles mapped to PartTypeX. Identical to NumPart_ThisFile                            |
| Flag_Cooling        | int32     |                                                      | Unused, set to 0                                                                                                                          |
| NumFilesPerSnapShot | int32     |                                                      |                                                                                                                                           |
| BoxSize             | double    | [/Header.BoundingBox](#header-boundingbox)           | Largest dimension of the bounding box                                                                                                     |
| Omega0              | double    | [/Cosmology.Omega_matter](#cosmo-omega_matter)       |                                                                                                                                           |
| OmegaLambda         | double    | [/Cosmology.Omega_lambda](#cosmo-omega_lambda)       |                                                                                                                                           |
| HubbleParam         | double    | [/Cosmology.h](#cosmo-littleH)                       |                                                                                                                                           |
| Flag_StellarAge     | int32     |                                                      | Unused, set to 0                                                                                                                          |
