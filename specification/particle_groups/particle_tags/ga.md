# Group Associated

Tag: `Group`

Includes:  [`Particles`](#tag-particles)

| Name         | Description                                                |
| ------------ | ---------------------------------------------------------- |
| GroupIDs | ID of the group the particles belong to |

This tag is agnostic to the type of grouping algorithm applied.
So examples of fields that could map to this include `fofgroup_ids`, `HaloID`, etc.
