# Massive

Tag: `Massive`

Includes:  [`Particles`](#tag-particles)

Particles with a mass.

| Name   | Type       | Description                      |
| ------ | ---------- | -------------------------------- |
| Masses | float[$N$] | Mass of the individual particles |

This tag does not need to be specified for fixed, uniform mass particles; however, we recommend still doing so, since the compressed Dataset would occupy minimal space and the user gains the benefit of the rich unit/description/etc metadata and uniform information access.
